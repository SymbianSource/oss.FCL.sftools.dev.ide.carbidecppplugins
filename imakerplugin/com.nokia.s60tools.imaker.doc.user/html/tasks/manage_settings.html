Managing settings

Before you can start building or flashing any ROM images, you must have at least one setting for one configuration. A configuration can be thought of as a container of product and other information. When you enter the ImageFlasher preferences via Window > Preferences... > ImageFlasher the configurations are automatically queried from iMaker and HWID and Target drop down menus are populated with data.

Creating a setting for a configuration

First select a product with a HWID from the HWID drop down menu. Then select the desired target from the Target drop down menu. Proceed with selecting the Image type and possible Flags. In the Additional parameters field you can enter additional build options if necessary. When done, click on the Create new configuration button to save the setting. Enter a name for your new setting and hit OK to save it.

Deleting a setting

The Settings drop down menu shows all the saved settings. If you wish to delete a setting first select it from the Settings drop down menu and then click on the Delete selected configuration
