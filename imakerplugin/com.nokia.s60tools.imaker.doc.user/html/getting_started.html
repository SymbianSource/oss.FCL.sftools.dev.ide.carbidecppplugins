Getting Started

The iMaker extension for Carbide.c++ is a tool for building ROM images. In the future also for flashing images to the end device. The information in this section will help you get started quickly using the basic features of iMaker.

Before you can create images with the iMaker, the following conditions must be met:

  • An active project in the Carbide.c++ workspace must exist. The project must be located on the top of a working build environment (an S60 3rd edition R&D SDK or higher)

Build ROM images by following these steps:

  1. Build image via iMaker Dialog.
  2. Build image via saved preferences.

While the image is being built, a build log can be seen in the console in Carbide.c++.
