iMaker build tool

iMaker is an image creation tool that provides a simple, standardized and configurable image creation framework. iMaker is based on the standardized Make system, which makes it platform-independent.

iMaker creates a FlashImage from a set of Symbian OS binary and data files. The tool mainly functions on top of the Symbian OS buildrom utility. The iMaker tool itself runs the Make tool and consists of a thin layer of Perl.
