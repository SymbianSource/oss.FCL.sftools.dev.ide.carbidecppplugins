Configuration

A configuration in iMaker can be thought of as a container of product information. It contains such information as the product name, hardware id (HWID), platform name, targets, and so on.

One configuration typically corresponds to one makefile on the file system level. This makefile may link to multiple other makefiles. iMaker gets its information mostly through configurations.
