Harmonized Test Interface (HTI)

HTI is a software component that enables remote controlling of Symbian OS. The purpose of HTI is to help and enable automated testing of devices running on Symbian OS.

On a high-level, the HTI on Symbian OS side consists of HTI Framework and plug-in Test Services. The HTI Framework handles the messaging and communication between the device doing the remote controlling (usually PC) and the Symbian OS device being controlled. The Test Services are components that are plugged in to the HTI Framework and they execute different functionalities. Examples of Test Services are key press emulation, screen shot capture, file transfer, process starting/stopping, etc.
