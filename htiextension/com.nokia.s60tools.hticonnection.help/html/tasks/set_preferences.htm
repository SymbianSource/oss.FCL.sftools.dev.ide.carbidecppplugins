Editing HTI connection settings

To edit the HTI connection settings:

  1. Select Window > Preferences > Carbide Extensions > HTI Preferences.
  2. In Use connection, do one of the following:
    • To edit an existing HTI connection, select a connection and click Edit.
    • To create a new HTI connection, click New.

    Follow the instructions in Setting up USB serial connection or Setting up Bluetooth connection.

    If you have problems with the connection, see Troubleshooting.

  3. Click Apply/OK.
