Editing HTI connection settings

To edit the HTI connection settings:

  1. Select Window > Preferences > Carbide Extensions > HTI Connection.
  2. In Use connection, do one of the following:
    • Select one of the exiting connections
      Note: If you select Current connection, HTI Connection will automatically use the currently active connection.If Current connection is selected, user can change the currently active connection easily either from Remote Connection view, or from a new Trim widget showing the current connection.
      • To edit an existing HTI connection, select a connection and click Edit.
      • To create a new HTI connection, click New.

    Follow the instructions in Setting up USB serial connection or Setting up Bluetooth connection.

    If you have problems with the connection, see Troubleshooting.

  3. Click Apply/OK.
