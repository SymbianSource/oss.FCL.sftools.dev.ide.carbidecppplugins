Viewing remote connections

You can view active remote connections in the Remote Connections view. To open the Remote Connections view, select Window > Show view > Other... > Carbide.c++ > Remote Connections.

Note that the Remote Connections view shows all connections in Carbide.c++, including connections configured for and used by other services. The HTI Connection Carbide.c++ Extension uses the generic Serial and TCP/IP connection types. HTI Connections use the following statuses:

  • Not connected
  • Trying to connect
  • Connected

In addition to viewing the connections status, you can do the following:

  • To edit an existing connection, doubleclick it in the Remote Connections view. For details, see Setting up USB serial connection and Setting up Bluetooth connection.
  • To delete a connection, rightclick it in the Remote Connections view and select Delete.

  • To set a correction as Current connection, rightclick it in the Remote Connections view and select Make Current.
