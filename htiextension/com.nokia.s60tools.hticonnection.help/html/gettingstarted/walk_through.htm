Basic walk-through

The HTI Connection view opens when a plug-in is started, that is, when another plug-in starts using the Harmonized Test Interface (HTI) service. You can open the HTI Connection also by selecting Carbide > HTI Connection.

You can start using the HTI Connection by taking the following steps:

  1. In the device, start the HTI Framework by selecting Menu > RnD Tools > HTI Admin > Options > Start.
  2. In Carbide.c++, start the HTI Connection by selecting Carbide > HTI Connection.
  3. (When starting HTI connection for the first time) Set up a connection to the device (via USB serial cable or Bluetooth).
  4. Start the Datagateway by clicking in the HTI Connection toolbar.
