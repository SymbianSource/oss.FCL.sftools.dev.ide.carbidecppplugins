Overview

The HTI Connection service provides a communication interface between Carbide.c++ and an S60 target device. The HTI Connection manages the connection between Carbide.c++ and the device by starting the HTI Datagateway and shows a generated log in the HTI Connection view. The HTI Connection view shows information of events and errors from Datagateway.exe. The connection parameters can be modified from HTI Preferences.
