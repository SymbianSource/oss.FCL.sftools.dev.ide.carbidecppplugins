Troubleshooting

The following table lists the messages that may be displayed when using the HTI Connection.

Message Action
HTI agent in the device is not responding.
  • Ensure that you have started the HTI framework in the device.
  • Ensure that your device is connected to a computer.
  • From computer settings, check which COM port is used.
  • Check the port and host fields (if using TCP/IP connection type).
Connection failed.
  • Select the correct COM port (if using serial connection type).
  • Check the port and host fields (if using TCP/IP connection type).
Warning: Testing of a connection will close the existing connection. You can resume the connection after the testing by clicking Cancel.
Datagateway has been shutdown unexpectedly. Do you want to manage connections?
  • Yes: Opens the HTI Connection preferences dialog.
  • No: Continues as before.
 Datagateway has been stopped by an unknown reason. It can be restarted either by clicking the Start Datagateway button in toolbar or by changing/testing settings in HTI Connection preferences.
