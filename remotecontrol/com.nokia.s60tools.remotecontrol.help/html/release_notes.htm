Product description

Remote Control provides an easy way of sharing device screen on a PC and controlling a device with a PC. Remote Control uses the HTI Connection service for communicating between Carbide.c++ and an emulator/device.

Remote Control can be started either by selecting Carbide > Remote Control, or from Window > Show View > Other... > Carbide Extensions > Remote Control.

Main features

  • The File Transfer mode allows to transfer files between PC and a target device.
  • The Screen Capture mode allows to take single screen shots.
  • The File Transfer mode contains Download as, Download and open, and Refresh commands.
  • The user can change the file name when downloading or uploading files.
  • Support for touch user interface.
  • Support for PC keyboard.
  • Support for multiple screenshots.
  • Support for virtual keyboards and key shortcut mappings.

What's new

  • Feature: The File Transfer mode contains Copy, Cut, Paste, and Rename commands.
  • Feature: Files and folders can be dragged and dropped to the File Transfer view.
  • Change: Modifications required to support Carbide.c++ 2.2
