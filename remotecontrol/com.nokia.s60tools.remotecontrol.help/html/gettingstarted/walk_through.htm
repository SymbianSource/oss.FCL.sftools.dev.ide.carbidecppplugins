Basic walk-through

You can start using Remote Control by taking the following steps:

  1. Start Remote Control by selecting Carbide > Remote Control, or Window > Show View > Other... > Carbide Extensions > Remote Control.
  2. Start the HTI Connection Carbide.c++ Extension by selecting Carbide > HTI Connection) if it is not already open, and set up an HTI connection. For more instructions on configuring the HTI connection, see the HTI Connection Help.
  3. Start the connection by clicking in the HTI Connection toolbar and selecting a connection to use.
  4. Save screenshots or transfer files. When the connection is established, the device screen is displayed in the Remote Control view. You can modify the settings of Remote Control view from Remote Control preferences.
