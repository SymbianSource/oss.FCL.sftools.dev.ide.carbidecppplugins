Using virtual keyboard

The RemoteControl Carbide.c++ Extension allows you to enter text and navigate in the Screen Capture mode using a virtual keyboard. To open the virtual keyboard, click the main menu of the Screen Capture mode, and select Keyboard.

There are two different keyboard layouts you can use: Simple and QWERTY. To switch between the keyboard layouts, click in the keyboard view toolbar.

Simple keyboard QWERTY keyboard

In addition, you can use a separate set of navigation keys by clicking in the keyboard view toolbar.
