Setting Remote Control preferences

To set the Remote Control preferences:

  1. In the Remote Control Carbide.c++ Extension, click , or select Window > Preferences...> Carbide Extensions > Remote Control Preferences.
  2. In the Screen Capture tab, set preferences for the Screen Capture mode:
    • In Delay before next refresh, select the refresh interval of the screen capture in milliseconds.
    • In Color mode, select the color mode for the screen capture. The available modes range from grayscale to 16 million colors.

    • In Save Screenshot Options, define a Name prefix and a Path for sequential screenshots. You can also define whether or not you want Remote Control to ask for a file name and location when saving single screenshots. If you clear the option, Remote Control will save single screenshots with the values entered in Name prefix and Path.

    • To configure PC keyboard shortcuts, click the Keys link at the bottom of the Screen Capture tab.
  3. In the File Transfer tab, set preferences for the File Transfer mode:
    • In Drives Shown, you can define the drives that are shown in the File Transfer view. By default, the drives c:, d:, e:, and z: are shown. To modify the list, use the Add and Remove buttons.
    • To view only the actually existing drives, select the option Get drive list from the device, if supported. Note that this feature works with HTI Framework versions 1.88, 1.89, 2.1 and newer.

    • In File Transfer Options, select whether or not you want to confirm a file replacement when uploading or downloading files, and whether or not you want to confirm a file or folder deletion. If you clear all the options, you will not be asked to confirm file replacement or file and folder deletion. Note that you can set this also in a file replacement or delete confirmation dialog.

  4. Click OK.
