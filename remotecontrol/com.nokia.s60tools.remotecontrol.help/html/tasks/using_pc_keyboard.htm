Using PC keyboard

The RemoteControl Carbide.c++ Extension allows you to enter text and navigate in the Screen Capture mode using your PC keyboard. The PC keyboard is activated when you enter the Screen Capture mode.

Currently, the following PC keyboard key presses are recognized and sent further to the target device or emulator:

  • Text characters: a-z, A-Z, öäå and ÖÄÅ.
  • Numeric characters: 0-9.

Other characters are sent as pure text and are supported depending on the current keyboard layout.

Customising PC keyboard shortcuts

There are no default shortcut mappings for non-textual characters, so you need to configure any hardware key shortcuts manually via Carbide.c++ Preferences.

  1. In Remote Control Carbide.c++ Extension, click and go to the Screen Capture tab.
  2. At the bottom of the tab, click the Keys link. This opens the shortcut key preferences view.
    You can also get to this view by selecting Window > Preferences > General > Keys.
  3. In the Command list, scroll down to the RemoteControl commands, and select a command you want set a shortcut to. You can set a shortcut to the following keys:
    • Up, down, left, and right navigation keys (North, South, West and East)
    • Selection (Center) key
    • Left software key
    • Right software key
    • Send key
    • End key
    • Application key
    • Clear key
    • Edit key
    • Grip open key
    • Grip closed key
    • Side (Voice) key
    • Power key
  4. In Binding, press the shortcut combination you want to use for the selected command.
    For example, for the Selection key (Center navi key) emulation, you may want to use the Enter key; place the mouse cursor in the Binding field, and press Enter.
  5. In When, select where you want to use the shortcut key combination. To restrict the shortcuts to the Screen Capture view of Remote Control, select In ScreenView.

  6. Configure the next key shortcut mapping, and when you are done, click OK.
