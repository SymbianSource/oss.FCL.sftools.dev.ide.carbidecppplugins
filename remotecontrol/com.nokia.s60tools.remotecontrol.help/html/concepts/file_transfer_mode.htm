File Transfer mode

The Remote Control File Transfer mode provides an easy way of transferring files between a device and a PC. The folder structure of the workspace root of Carbide.c++ is displayed in the File Transfer view after you have started the Datagateway.

For the file transfer mode preferences, see Setting Remote Control preferences.
