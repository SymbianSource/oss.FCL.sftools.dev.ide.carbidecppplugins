Overview

The Symbian Foundation Templates Wizard Carbide.c++ Extension is a collection of additional wizards for Carbide.c++.

You can access the templates in Carbide.c++ by selecting File > New > Other > Symbian Foundation Templates.

You can access the templates also by right-clicking an MMP file in Symbian Project View Navigator or in Project Explorer.
