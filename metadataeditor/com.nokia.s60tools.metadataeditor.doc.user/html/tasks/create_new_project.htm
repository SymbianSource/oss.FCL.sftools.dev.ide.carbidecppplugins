Creating a new project

Before you start

Before you can work with metadata files, you must have a project in the Carbide.c++ environment and a folder that contains the metadata files. If you already have a project that you want to use, see the instructions in Importing a project.

To create a new project from scratch, do the following:

  1. Go to File > New > Project...
  2. Select General > Project, and click Next.
  3. Type in the name of your project and select the directory where it is stored.
  4. Click Finish.

    The new project is created to the Carbide.c++ project view.

  5. To create a folder for the metadata, right-click the project name and select New > Folder.
  6. Enter a name for the metadata folder.
  7. Click Advanced and select the Link to folder in the file system option.
  8. Click Browse to select your project folder that contains the API Metadata.
  9. Click Finish.
